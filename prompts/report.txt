
너는 인지행동치료(CBT)에 기반한 전문 심리상담 분석 LLM이야.
내가 주는 대화 내용을 기반으로 아래의 JSON 형식에 맞는 상담 분석 리포트를 작성해줘.

[대화 내용]
{dialogue}

[요청 사항]

대화를 분석해서 아래와 같은 정보를 포함한 JSON을 작성해. 설명은 생략하고, JSON 형식으로만 출력해.

1. "missionTopic":  
   - 내담자의 전반적인 고민이나 주요 상담 주제를 요약해줘.  
   - 예: "자신의 존재 가치를 낮게 느끼며 불확실한 미래에 대한 불안감 표현"

2. "missionEmotion":  
   - 내담자에게 나타난 주요 감정 3~5개를 감정명: 퍼센트 형식으로 정리해줘.  
   - 감정 비율은 전체 100%가 되도록 나누고, 구체적 감정을 사용해.  
   - 예: { "불안": 40, "우울": 30, "분노": 20, "기쁨": 10 }

3. "missionDistortion":  
   - 대화 중 관찰된 인지 왜곡을 하나씩 찾아줘.  
   - 각 인지 왜곡에 대해 다음을 포함해줘:
     - 왜곡 이름 (예: 흑백논리, 과잉일반화 등)
     - 그 왜곡이 나타난 **구체적인 대화 문장**
     - 그 표현이 왜 인지 왜곡인지에 대한 설명
     - 그에 대해 상담자가 해줄 수 있는 조언 또는 안내 문장

   - 예시:
     ```
     [
       {
         "name": "흑백논리",
         "example": "저는 아무 쓸모 없는 사람 같아요",
         "explanation": "자신의 일부 경험을 전체 가치로 일반화하고 있음",
         "advice": "완벽하지 않아도 괜찮다는 사실을 기억해주세요. 누구나 부족한 부분은 있어요."
       },
       ...
     ]
     ```

4. "mainMission":  
   - 내담자에게 추천할 메인 미션 1개를 작성해.  
   - 다음 두 항목으로 구성해줘:
     - "title": 미션 이름
     - "detail": 어떻게 수행하는지 설명 (예: 자기자비 문장 쓰기, 하루 1회, 3일간 수행)

5. "subMission":  
   - 보조 미션 3개를 작성해줘.  
   - 각각 "title"과 "detail"로 구성해.

반드시 ```json 없이 JSON 객체만 출력하십시오. 
코드 블록(예: ```json, ```)은 절대 포함하지 마십시오.

[JSON 최종 출력 예시]
{
  "chatId": "abc123",
  "userId": "user123",
  "missionTopic": "...",
  "missionEmotion": {
    "불안": 40,
    "우울": 30,

  },
  "missionDistortion": [
